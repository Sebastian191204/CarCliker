# CarClicker - MVC

## Descripción

CarClicker es una aplicación web interactiva desarrollada con arquitectura MVC (Modelo-Vista-Controlador) que permite a los usuarios explorar y votar por sus autos favoritos. La aplicación presenta una interfaz intuitiva con una barra lateral que muestra diferentes modelos de autos clásicos, permitiendo al usuario navegar entre ellos y votar por sus preferidos.

### Características principales:
- **Interfaz responsiva**: Diseño moderno con gradientes y animaciones suaves
- **Arquitectura MVC**: Separación clara entre modelo, vista y controlador
- **Persistencia de datos**: Los votos se guardan localmente usando localStorage
- **Ranking en tiempo real**: Muestra el ranking de autos más votados
- **Navegación intuitiva**: Click en cualquier auto de la lista para ver su imagen

### Tecnologías utilizadas:
- HTML5
- CSS3 (con gradientes y animaciones)
- JavaScript (ES6+ con clases)
- Arquitectura MVC
- localStorage para persistencia

## Rama principal
La rama principal del proyecto es `master`.

## Cómo ejecutar la aplicación

### Opción 1: Ejecutar localmente
1. Clona el repositorio:
   ```bash
   git clone https://github.com/Sebastian191204/CarCliker.git
   cd CarCliker
   ```

2. Abre el archivo `index.html` en tu navegador web preferido.

### Opción 2: Usar un servidor local
Para una mejor experiencia, puedes usar un servidor local:
```bash
# Con Python (si tienes Python instalado)
python -m http.server 8000

# O con Node.js
npx http-server

# Luego abre http://localhost:8000 en tu navegador
```
